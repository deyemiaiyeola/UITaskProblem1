-Supported browsers: Google chrome (default), Firefox, MS edge

-Prerequisites for running the application: Java 11 or higher

-Maven 3.6 or higher

-Git


Clone the repository using the git clone command  [INSERT GIT REPO HERE]


-Create test file directory and sample  (using linux, the command is the following )filemkdir -p src/test/resources/testfilesecho "This is a test file for upload testing a form field!“ > src/test/resources/testfiles/testfile.txt

-Build the project using the command: mvn clean compile

-Run the tests using the command: mvn test

-Viewing Test ReportsAfter the tests complete, you can find the test reports in the following locations: target/cucumber-reports/cucumber.html

-Open this file in any web browser to view the report

-There is also the option of a JSON Report which is located on the following directory: target/cucumber-reports/cucumber.json

-This file can be used for integration with other reporting tools 

-Surefire report location: target/surefire-reports/ which contains an HTML report as well as an XML report of the tests that passed.


Note: If you encounter any WebDriver errors, ensure you have the appropriate browser driver installed.
